# 示例：没有找到客户时怎么恢复

## 用户输入

```text
我卖企业税务咨询，帮我在快手找客户。
```

## 可能结果

工具返回评论数量少，或评论大多是泛讨论，没有明显购买意向。

## 正确输出

```text
这轮没有筛出高意向客户。我在快手按「企业税务咨询、公司节税、老板报税」看了 5 条内容，共读到 61 条公开评论，但大部分是泛讨论或同行科普，没有出现明确的“想咨询、怎么收费、公司遇到问题”这类购买信号。

我建议下一轮不要继续硬刷这 3 个词，先换成更具体的痛点词：
1. 小规模纳税人补税怎么办
2. 公司被税务稽查怎么办
3. 老板公转私风险

如果你有具体服务城市，也可以加上城市词，这样更容易找到真实咨询需求。
```

## 为什么这个示例有效

- 没有把空结果包装成成功。
- 解释了看过多少内容和为什么没找到。
- 给出下一轮关键词，不让用户自己摸索。
- 避免平台额度被无效关键词浪费。
